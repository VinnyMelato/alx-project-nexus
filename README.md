Taxi APP
A simple taxi booking application built with Python and Django.

An Overview  
This project is a web-based taxi application that allows users to request rides and drivers to accept them. It was built as part of the ALX Project Nexus, demonstrating full-stack web development concepts using Django. The goal is to provide a basic ride-hailing experience similar to popular platforms.  

Features
- User registration and login  
- Request a taxi to a specific destination  
- Driver view to accept available rides  
- Simple booking flow and ride management  
- Django backend with views and routing  

Tech Stack
This project uses:
- Python  
- Django  
- HTML/CSS  
- SQLite (default Django database)
